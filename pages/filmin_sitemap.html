Sitemap

A la hora de realizar el sitemap se desarrollaron dos diferentes. Uno de ellos es un sitemap para el total de la web de Filmin (todo lo que corresponde abarcar en nuestro proyecto) y el otro sitemap esta dedicado exclusivamente a las funciones de las listas. Esto se debe a que en un primer momento, para analizar la web de filmin quisimos desarrollar un sitemap para entender y visualizar mejor la página. El sitemap de listas llega mas tarde para incorporar nuevas funcionalidades a la página y darle nuevos usos a la app.
